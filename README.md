# Luis Calderon - Job Search Command Center

**90-Day Job Search Dashboard** for VP/Director/Principal PM roles with AI-powered tools.

## Target Role
- VP Product / Director of Product / Head of Product / Principal PM
- $200K+ base compensation
- San Diego (preferred), Remote, LA/SF
- AI/ML product leadership focus

## Dashboard Features

### Core Tracking
- **Real-time progress tracking** - Days elapsed, applications sent, interviews scheduled
- **4-phase timeline** - Setup > Volume > Interviews > Close
- **Weekly metrics tracker** - Applications, outreach, calls, interviews
- **Auto-save** - All progress saved to browser localStorage + optional cloud sync

### CRM & Contacts
- **Contact management** - Track recruiters, hiring managers, network connections
- **Conversation history** - LinkedIn and email message tracking
- **Message templates** - Quick responses for common outreach scenarios
- **Duplicate detection** - Smart merging of duplicate contacts

### AI-Powered Tools
- **AI Message Generator** - Context-aware outreach messages
- **LinkedIn Post Generator** - 6 post type templates (Tool Review, Insight, Framework, etc.)
- **Outreach Optimizer** - AI-suggested improvements to messages

### Thought Leadership
- **Daily action tracking** - Comment, engage, learn routine
- **Weekly content plan** - Mon/Wed/Fri posting schedule
- **Monthly goals** - Network growth and engagement targets
